# Model Asset Attribution

MoveBeta vendors the MoveNet SinglePose Lightning TensorFlow.js graph and weight shards for same-origin, cache-first browser analysis.

- Source provider: TensorFlow Hub
- Source model: MoveNet SinglePose Lightning
- Source model URL: https://tfhub.dev/google/tfjs-model/movenet/singlepose/lightning/4/model.json?tfjs-format=file
- Vendored manifest: `public/model-assets.json`
- Vendored files: `public/models/movenet/singlepose/lightning/4/`
- Provenance refresh: `npm run model:assets:provenance`

The generated manifest records file paths, byte sizes, SHA-256 digests, source URLs, and the generation timestamp. Keep this notice with release evidence whenever model assets are distributed.

License review is required before commercial distribution. The provenance doctor proves source, integrity, attribution notice presence, and share-safety; it does not infer final legal clearance from downloaded model files.
